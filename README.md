# Music Web App

A responsive music web app built with Next.js, TypeScript, and an incredible Aceternity-powered UI.

## Tech Stack
- Next.js
- TypeScript
- Aceternity UI
- Tailwind CSS or CSS

## Features
- Modern and clean UI
- Smooth animations
- Fully responsive design
- Fast performance
- Reusable components

## Clone the Repository
Use the command below to clone the project.

```bash
git clone https://github.com/Abdul-Wahab08/music.git
```
##Setup and Run

```bash
cd music
npm install
npm run dev
```

##Open your browser and visit:

```bash
http://localhost:3000
